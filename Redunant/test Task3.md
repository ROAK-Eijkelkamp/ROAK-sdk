# ROAK SDK Borehole Data Task (Hard)

Hi!  

Your assignment is to use the ROAK Python SDK to fetch the max depth (biggest) data from a specific project and display the results.

## Goal

- Fetch all boreholes from the project named `Demo boreholes`.
- Retrieve data for each borehole for a recent time range.
- Filter the data so only the borehole with the largest depth is printed
- Print or log the fetched data.

## What you have

- **User Guide**: explains how to set up the SDK, authenticate, and lists the available methods.  
- **Starter Python file**: empty file where you will write your code.

## Requirements


1. Fetch all assets of type **borehole** within that project (`Demo boreholes`).
2. Retrieve the data for each borehole for a recent period.
3. Print the output in any readable form.

> Notes:
> - python has build in a method for getting the **largest** value out of dataset or list
> - You do **not** need to hardcode any client or authentication details beyond what the User Guide allows.  
 

Good luck! Take notes on any points that are confusing or unclear in the SDK—you’re helping improve the documentation while completing the task.
